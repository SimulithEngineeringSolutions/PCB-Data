# Arduino Hat Material Viewer

This folder keeps a single supported viewer for the current goal:

- [arduino_hat_stackup_vedo.py](/c:/Users/Ryant/Documents/GitHub/PCB%20Data/viewer/arduino_hat_stackup_vedo.py)

It renders a simplified 3D material model of the `Arduino hat` PCB with:

- fiberglass / dielectric layers
- copper traces
- copper pads
- vias
- board outline

## Input data

The viewer reads:

- `../output/arduino_hat/copper_paths.json`

That file is generated by:

- [export_kicad_copper_paths.py](/c:/Users/Ryant/Documents/GitHub/PCB%20Data/export_kicad_copper_paths.py)

## Run

1. Export the board routing and stack data with KiCad's Python:

```powershell
& "C:\Program Files\KiCad\10.0\bin\python.exe" .\export_kicad_copper_paths.py
```

2. Open the 3D material viewer:

```powershell
python .\viewer\arduino_hat_stackup_vedo.py
```

## Notes

- This is now the only maintained viewer in this repo for the Arduino hat material model.
- The geometry is intentionally simplified for responsiveness.
- Layer spacing is inferred from board thickness and active copper layers, so this is a visual stackup model rather than a manufacturing-grade laminate definition.
