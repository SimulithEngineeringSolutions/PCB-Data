# Arduino Hat Material Viewer

This repo keeps a single supported viewer for the current goal:

- [modules/arduino_hat_stackup_vedo.py](/c:/Users/Admin/Documents/Github/PCB-Data/modules/arduino_hat_stackup_vedo.py)

It renders a simplified 3D material model of the `Arduino hat` PCB with:

- fiberglass / dielectric layers
- copper traces
- copper pads
- vias
- board outline

## Input data

The viewer reads:

- `output/arduino_hat/copper_paths.json`

That file is generated by:

- [modules/export_kicad_copper_paths.py](/c:/Users/Admin/Documents/Github/PCB-Data/modules/export_kicad_copper_paths.py)

## Run

1. Run the main pipeline and choose a `.kicad_pcb` file from the file picker:

```powershell
python .\pcb_data_pipeline.py
```

2. If you want to run the extractor directly with KiCad's Python:

```powershell
& "C:\Program Files\KiCad\10.0\bin\python.exe" .\modules\export_kicad_copper_paths.py
```

3. Open the 3D material viewer directly:

```powershell
python .\modules\arduino_hat_stackup_vedo.py
```

## Notes

- This is now the only maintained viewer in this repo for the Arduino hat material model.
- The geometry is intentionally simplified for responsiveness.
- Layer spacing is inferred from board thickness and active copper layers, so this is a visual stackup model rather than a manufacturing-grade laminate definition.
